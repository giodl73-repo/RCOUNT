# RCOUNT Wave

Use this skill to create or update RCOUNT waves.

## Wave requirements

- Define a product-neutral count/audit package goal.
- Identify affected crates and consumer repos.
- Break the wave into small pulses.
- Include validation expectations for every pulse.
- Record non-goals so RCOUNT does not absorb BISECT application logic.
